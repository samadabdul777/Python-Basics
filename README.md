# Student Marks Project

Hi! This is a small project I made to keep track of students and their marks.

## What it does
- Lets you add student names and their marks
- Calculates the average marks
- Finds the student with highest marks
- Shows all the students and their marks

## How to run
1. Make sure Python is installed
2. Open this file in your terminal or command prompt
3. Run:
   python student_marks.py
4. Follow the instructions to add students and their marks
